Key Components Explained

Residual Blocks (Skip Connections)

Allow gradients to flow directly through the network. Each block learns residual functions F(x) = H(x) - x, enabling 50+ layer depth.

Batch Normalization

Normalizes activations in each mini-batch, reducing internal covariate shift and allowing higher learning rates.

Transfer Learning

Pre-trained on ImageNet (1.2M images, 1000 classes) then fine-tuned on medical imagery for robust feature extraction.

Dropout Regularization

30% dropout in FC layers randomly deactivates neurons during training to prevent overfitting.

Key Components Explained

Encoder Path (Contracting)

Progressive downsampling captures context and "what" is in the image. Each level doubles feature channels while halving spatial dimensions.

Decoder Path (Expanding)

Progressive upsampling recovers spatial resolution and "where" objects are located. Reconstructs the segmentation mask at full resolution.

Skip Connections (Concatenation)

Bridge encoder and decoder at each level. Preserves fine-grained spatial details that would otherwise be lost during downsampling.

Residual Blocks

Enable training of deeper networks by allowing gradients to flow directly. Improves convergence and final segmentation accuracy.

Why Focal Tversky Loss?

Handling Class Imbalance in Medical Imaging

The Challenge

Brain tumor regions typically occupy only 2-10% of the total image area. Standard loss functions like Binary Cross-Entropy would be dominated by the large background class.

The Solution

Focal Tversky Loss with α=0.7, β=0.3 penalizes false negatives more heavily, ensuring the model doesn't miss small tumors while maintaining precision.

Tversky Index Formula:
TI = TP / (TP + α·FN + β·FP)
Where: TP = True Positives, FN = False Negatives, FP = False Positives
